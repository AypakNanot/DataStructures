- [稀疏数组](sparse-array/src/main/resources/SparseArray.md)
 
将常规的二维数组转换为稀疏数组，来节省空间。
这个二维数组大多数值都是默认值(或者为0)的情况下使用，才不至于反而增加空间。场景应用：存储五子棋，存储残局象棋等
